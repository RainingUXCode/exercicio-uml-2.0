# 📚 Lista de Exercícios: Modelagem UML (Versão 2.0)

Este repositório centraliza a resolução da segunda lista de exercícios práticos propostos. O objetivo principal deste projeto é traduzir modelos estruturais (Diagramas de Classes UML, Casos de Uso e Modelos Relacionais) e requisitos de sistema em aplicações web reais e interativas.

---

## 🛠️ Tecnologias Base

Todos os projetos desta lista foram desenvolvidos utilizando o ecossistema moderno de frontend:

- **React** (via Vite)
- **TypeScript** / **JavaScript**
- **Mantine UI** & **Tailwind CSS** para estilização e responsividade

## 🖥️ Modelagens Inclusas na Raiz

Além das aplicações em código, os arquivos de modelagem unificados encontram-se disponíveis para verificação direta no diretório principal:

- **`Exercício 2 de Análise e Projeto de Sistemas.docx`**: Arquivo Word contendo classes, atributos, métodos e as imagens dos outros diagramas.
